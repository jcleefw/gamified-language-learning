# Create Epic Plan
1. Identify context and next EP number
2. Gather requirements (Why, What, Scope)
3. Identify dependencies (`Depends on`) and parallelism (`Parallel with` — epics with the same dep that can run concurrently)
4. Determine if the epic has distinct sub-domains — if so, define Phase groupings (EP##-PH01, EP##-PH02) for the Stories section
5. Generate EP using EP-TEMPLATE.md in .agents/plans/
6. Present to user for approval
