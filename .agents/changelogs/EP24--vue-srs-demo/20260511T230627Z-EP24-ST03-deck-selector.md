## Implementation

Renders two sections:

1. **Resume banner** — shown only when `loadSession()` returns non-null. Displays the saved `deckId` and two buttons: "Resume session" (`emit('resume')`) and "Clear & start over" (`emit('clear')`).
2. **Deck list** — iterates `mockDecks` from `@gll/srs-engine-v2/data/mock/mock-decks`. Each deck button shows the topic and word count, emits `select(deckId)` on click.

Emits: `select(deckId: string)`, `resume`, `clear`.

## Decisions

- `loadSession()` called at component setup time (not reactive) — the banner state is determined once on mount, which is sufficient since this screen only shows when there is no active session.
- Mock decks imported directly from the engine subpath export rather than passed as props — they are static seed data, not runtime state.
