## Implementation

Emits: `next`.

Renders:
- Score header: `"N / M correct"`
- Table columns: Word | Seen | Correct | Mastery | Streak ✓/✗
- Mastery column uses 5 dot indicators (filled green up to `state.mastery`) plus the numeric value
- Word IDs strip the `th::` prefix for readability
- "Next Batch →" button emits `next`

`BatchSummary` interface is exported from this component so `App.vue` can reuse the type.

## Decisions

- 5 mastery dots hard-coded to match `maxMastery: 5` default rather than deriving from config — simpler for a demo; config is not passed as a prop.
- `th::` prefix stripped inline with `replace('th::', '')` — sufficient for mock data; no need for a more general ID parser.
