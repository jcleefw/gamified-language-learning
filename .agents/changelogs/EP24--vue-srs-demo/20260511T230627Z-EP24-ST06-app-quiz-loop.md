## Implementation

Single-file screen controller. No router. `screen: ref<'select' | 'quiz' | 'results'>` drives which component is rendered.

**Config** (hardcoded):
```ts
{ questionLimit: 8, masteryThreshold: 2, streakThresholds: { correctStreakThreshold: 2, wrongStreakThreshold: 2, maxMastery: 2 } }
```

**Engine call sites**:

| Function | Where called |
|---|---|
| `nextActivePool` | `initSession` — seeds active pool from deck words on first load |
| `composeBatchMulti` | `startBatch` — generates up to 8 questions per batch |
| `updateMasteryState` | `onAnswered` — after final answer in batch |
| `nextActivePool` | `onAnswered` — after mastery update, retires mastered words |
| `saveSession` | `onAnswered` — persists state immediately after pool rotation |

**Session flows**:
- `onMounted`: calls `loadSession()`; if found, stays on `'select'` so `DeckSelector` shows the resume banner
- `onResume`: hydrates all state refs from saved session, calls `startBatch()`
- `onSelect`: calls `initSession(deckId)` — builds fresh pool and starts batch
- `onClear`: calls `clearSession()`, resets all refs, returns to `'select'`

**Batch completion** (`onAnswered` when `currentIndex === questions.length`):
1. `updateMasteryState(answers, runState, prevState, recheckPending, recheckReentered, ...)`
2. `nextActivePool(active, queue, ...)` — retires mastered words, fills from queue
3. `saveSession(...)` — persists updated state
4. Computes `batchScore` and `summary` from `runState`
5. Sets `screen = 'results'`

## Decisions

- `prevState` snapshot taken as `new Map(runState.value)` before calling `updateMasteryState` — required by the engine to detect newly mastered words this batch.
- `onMounted` leaves `screen = 'select'` when a saved session exists rather than jumping straight to `'quiz'` — the user should explicitly choose to resume, consistent with the resume banner UX.
- `recheckReentered` passed as `recheckExempt` to `nextActivePool` — keeps re-entered recheck words in the active pool even if their mastery count would otherwise retire them.
